# Exploratory Data Analysis: Pokemon
Final project for `CSE 2600: Intro to Data Science`

## Preamble
The datasets are acquired from https://play.pokemonshowdown.com/data/sets/, the **showdown_preprocessor** tool was used to take their data and reformat it to better suit our usage.

If you find data from another source that is more up to date at some point in the future, create a new preprocessor that takes your input database and outputs a file that matches our expected input for the main processor.

## How to Run
If the data you're using has already been preprocessed (or you're using ours), simply running the **processor** is all that's needed to analyze the data.
